How to Change Your Trend Micro Activation Code on Any Device

For Windows or Mac:

  1. Open the Trend Micro Security application.
  2. Click Subscription Information or Expiry Date.
  3. Select Change Activation Code.
  4. Enter the new code and confirm.

For Android or iOS:

  1. Open the Trend Micro Mobile Security app.
  2. Tap the menu (☰) → Account → Subscription.
  3. Select Update Activation Code.
  4. Enter the new key and tap OK.

Activate Trend Micro Subscription Purchased from Retail Stores

  1. Scratch off the label to reveal your activation key.
  2. Go to www.trendmicro.com.au/activate.
  3. Sign in or create your account.
  4. Enter your activation code and click Next.
  5. Download and install your software.

How to Activate Trend Micro ID Protection

  1. Visit www.trendmicro.com.au/activate.
  2. Sign in to your Trend Micro account.
  3. Enter your ID Protection activation code.
  4. Download and install the ID Protection app.
  5. Sign in and enable identity monitoring and alerts.

FAQs — Trend Micro Activation Help

1. Can I use the same activation code on multiple devices?

Yes. Depending on your plan, Trend Micro Maximum Security supports multiple devices.

2. My activation code isn’t working. What should I do?

Ensure you enter the code correctly without spaces and confirm you’re using the correct region site (.au for Australia). If it still fails, contact Trend Micro Support.

3. How do I renew my expired subscription?

Sign in to your Trend Micro account, select your product, and click Renew Now. A new activation code will be emailed after payment.
